A self-hosted MQTT environment for IoT sensor data from ESP8266 dev boards

This repository contains the code that I use for my IoT gadgets at home. I have a number of ESP8266 boards with sensors that publishes data via MQTT to a Raspberry Pi. The RPi stores the data in a MongoDB and forwards the data to a cloud service (Adafruit IO).

Details
